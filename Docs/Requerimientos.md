# Requerimientos del Proyecto

## Requerimientos Funcionales
1.  **Control de Encendido:** El sistema debe tener un botón dedicado para encender la bombilla.
2.  **Control de Apagado:** El sistema debe tener un botón dedicado para apagar la bombilla.
3.  **Visualización:** La bombilla debe emitir un color "Verde Neón" cuando esté encendida y un estado apagado/opaco cuando esté inactiva.

## Requerimientos No Funcionales
1.  **Diseño (UI/UX):**
    - Estilo visual: **Claymorphism** (formas redondeadas, sombras suaves, aspecto 3D "arcilla", colores amigables).
    - Paleta de colores: Amigable (tonos pastel, bajo contraste en fondos, alto contraste en elementos activos).
2.  **Adaptabilidad:** La aplicación debe ser **Responsive**, funcionando correctamente en dispositivos móviles y tablets.
3.  **Calidad de Código:**
    - Código limpio y eficiente.
    - Buenas prácticas de programación.
    - Documentación completa mediante **JSDoc** en los archivos JavaScript.

## Restricciones Técnicas
- Tecnologías: HTML5, CSS3, JavaScript (Vanilla).
- Navegadores: Compatibilidad con navegadores modernos (Chrome, Firefox, Edge, Safari).
